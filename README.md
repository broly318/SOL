# このプラグインについて
このプラグインは、漫画「AKIRA」に登場する衛星兵器「SOL（Satellite in Orbital Laser-weapon）」をマインクラフトで再現したものになります。
# 使い方
本６４個を持って右クリック、または左クリックで目線の先にあるブロック（最大１００メートル先まで）。ロックオンされた地点にレーザーがおちます。
